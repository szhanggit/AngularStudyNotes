Creating a Two-Way Binding on the Host Element:
src\app\twoway.directive.ts

The first argument to the @HostListener decorator specifies the name of the event that will be handled by the listener. 
The second argument is an array that will be used to provide the decorated methods with arguments.
@HostListener("input", ["$event.target.value"])
updateValue(newValue: string) {
    this.fieldValue = newValue;
    this.update.emit(newValue);
}
