It is for injecting variable value into a component.
1. The injectionToken is defined in tokens.ts.
export const MY_TOKEN = new InjectionToken<string>('myToken');
2. Register it in providers with token value.
  providers: [{provide: MY_TOKEN, useValue: 'Token Value'}],
3. Inject it into component:
  constructor(@Inject(MY_TOKEN) private myToken: string) 
  {
    console.log("Injected myToken", myToken);
  }
